# Claude Code 终极指令手册 (v2.1.72+) 🚀

这是一份结合了基础管理与最新进阶功能的 Claude Code 全能指南。

---

## 🛠️ 进阶四招：掌控全局 (v2.1.72+ 专属) 
*如果提示 "Unknown skill"，请执行 `npm install -g @anthropic-ai/claude-code`或 claude update`更新 没成功的话加上sudo 。*

- **`/simplify` (代码体检)**：让 AI 给代码做“深度扫描”，寻找重复逻辑、优化空间及死代码。
- **`/rewind` (战术回滚)**：AI 跑偏了？双击 `Esc` 或输入此命令，精准回退到之前的任何 Checkpoint。
- **`/btw` (随口插话)**：主任务执行中突然有个小疑问？用 BTW 提问，不占用上下文，不干扰主线思维。
- **`/insights` (战盘复盘)**：复盘过去 30 天的开发记录，分析你的行为模式并提供提效建议。

---

## 🧩 基础管理：稳扎稳打

### 1. 记忆与上下文
- **`/memory`**：编辑或追加 AI 的长期记忆。配合 `#` 命令，让 Claude 记住你的特定习惯。
- **`/compact`**：当上下文过长时主动进行压缩，避免关键信息被遗忘。
- **`/clean`**：大脑清理！结束当前复杂任务后清空堆栈，重置运行环境。

### 2. 项目感知
- **`/init`**：全量扫描项目，生成或更新 `CLAUDE.md`。这是 Claude 的“项目白皮书”。
- **`/mcp`**：管理和查看当前所有的扩展工具（Model Context Protocol）连接状态。

---

## 📊 效率工具与更新
- **实时监控**：在终端运行 `ccusage blocks --live` 实时查看 Token 消耗。
- **版本更新**：输入 `/update` 自动同步到社区最新版本。

---

## 💡 最佳实践之 CLAUDE.md
在项目根目录维护好这个文件，它应包含：
1. **项目背景**：让 AI 知道它在为什么业务服务。
2. **技术栈**：明确框架版本。
3. **架构设计**：阐明模块间的调用逻辑。
4. **编码规范**：代码怎么命名、怎么注释的“家法”。
