# OpenClaw 接入微信

OpenClaw（原 Clawdbot）是一个开源、本地优先的 AI 代理网关，支持直接操作电脑、浏览网页、执行命令。通过微信插件接入，可以实现与 AI 代理的无缝对话。

---

## 个人微信一键接入（ClawBot 插件）

此方案适用于 **iOS 8.0.70+** 版本的个人微信。

### 📋 前置条件

*   **微信版本**：iOS 8.0.70 及以上。
*   **OpenClaw 实例**：已在本地或服务器环境部署并运行。
    *   *若未安装，可执行：`npm install -g openclaw@latest`*

### 🚀 操作步骤

#### 1. 开启微信插件
打开手机微信，依次进入：**我** -> **设置** -> **插件**。

#### 2. 查找 ClawBot
在插件列表中找到 **「ClawBot」** 卡片并点击进入。在页面中你可以看到相关的安装说明。

#### 3. 执行安装命令
在运行 OpenClaw 的终端（服务器或本地电脑）中，复制并执行以下命令：

```bash
npx -y @tencent-weixin/openclaw-weixin-cli@latest install
```

#### 4. 扫码绑定
命令执行后，终端会生成一个 **二维码**。使用当前的微信扫码进行身份绑定。

#### 5. 确认部署成功
*   **终端反馈**：扫码成功后，终端会提示连接成功并自动重启 OpenClaw。
*   **后台查看**：登录 OpenClaw 管理后台，进入 **「频道 (Channels)」** 页面，确认微信图标已亮起且状态为正常。

#### 6. 开始对话
现在，你已经可以直接在微信聊天窗口中发送消息，与你的 OpenClaw 智能体进行交互了。
